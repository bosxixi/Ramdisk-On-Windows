================================================================================
MICROSOFT 基础类库: driverSetup 项目概述
===============================================================================

应用程序向导已为您创建了这个 driverSetup 应用程序。此应用程序不仅演示 Microsoft 基础类的基本使用方法，还可作为您编写应用程序的起点。

本文件概要介绍组成 driverSetup 应用程序的每个文件的内容。

driverSetup.vcxproj
这是使用应用程序向导生成的 VC++ 项目的主项目文件。 
它包含生成该文件的 Visual C++ 的版本信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

driverSetup.vcxproj.filters
    这是使用“应用程序向导”生成的 VC++ 项目筛选器文件。
    它包含有关项目文件与筛选器之间的关联信息。在 IDE 中，通过这种关联，在特定节点下以分组形式显示具有相似扩展名的文件。例如，“.cpp”文件与“源文件”筛选器关联。

driverSetup.h
这是应用程序的主要头文件。它包括其他项目特定的头文件(包括 Resource.h)，并声明 CdriverSetupApp 应用程序类。

driverSetup.cpp
这是包含应用程序类 CdriverSetupApp 的主要应用程序源文件。

driverSetup.rc
这是程序使用的所有 Microsoft Windows 资源的列表。它包括 RES 子目录中存储的图标、位图和光标。此文件可以直接在 Microsoft Visual C++ 中进行编辑。项目资源位于 2052 中。

res\driverSetup.ico
这是用作应用程序图标的图标文件。此图标包括在主要资源文件 driverSetup.rc 中。

res\driverSetup.rc2
此文件包含不在 Microsoft Visual C++ 中进行编辑的资源。您应该将不可由资源编辑器编辑的所有资源放在此文件中。


/////////////////////////////////////////////////////////////////////////////

应用程序向导创建一个对话框类:

driverSetupDlg.h，driverSetupDlg.cpp - 对话框
这些文件包含 CdriverSetupDlg 类。该类定义应用程序主对话框的行为。该对话框的模板位于 driverSetup.rc 中，该文件可以在 Microsoft Visual C++ 中进行编辑。


/////////////////////////////////////////////////////////////////////////////

其他标准文件:

StdAfx.h，StdAfx.cpp
这些文件用于生成名为 driverSetup.pch 的预编译头 (PCH) 文件和名为 StdAfx.obj 的预编译类型文件。

Resource.h
这是标准头文件，它定义新的资源 ID。
Microsoft Visual C++ 读取并更新此文件。

driverSetup.manifest
	应用程序清单文件供 Windows XP 用来描述应用程序
	对特定版本并行程序集的依赖性。加载程序使用此
	信息从程序集缓存加载适当的程序集或
	从应用程序加载私有信息。应用程序清单可能为了重新分发而作为
	与应用程序可执行文件安装在相同文件夹中的外部 .manifest 文件包括，
	也可能以资源的形式包括在该可执行文件中。 
/////////////////////////////////////////////////////////////////////////////

使用说明：

将对应版本的ramdisk.sys, ramdisk.inf, coinstallor.dll放在对应文件夹下，将对应32或64位devcon.exe放在对应文件夹下

先安装,再配置

注意避免重复安装
